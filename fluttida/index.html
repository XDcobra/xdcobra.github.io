Fluttida

A small toolkit and example apps for analyzing, instrumenting, and testing Flutter applications.

What is Fluttida?

Fluttida provides examples, scripts, and tooling aimed at developers and researchers who need to inspect, instrument, or test Flutter apps. It bundles practical examples for hooking, network interception, and runtime inspection useful during security assessments, debugging, or research.

Use cases

  • Security testing and reverse-engineering of Flutter Android/iOS apps.
  • Runtime instrumentation using Frida scripts and example hooks.
  • Network interception and TLS/debugging helpers for development.
  • Educational examples for how Flutter apps call native APIs.
  • Rapid prototyping of app-analysis workflows and tooling.

Getting started

  1. Clone the repository and read the example app README: example_app/fluttida/README.md.
  2. Open the example app in example_app/fluttida and follow the README to run the app and attached tools.
  3. Inspect and reuse the included Frida helpers:
    • HTTP Stack detection scripts (frida_detect_engine)
    • HTTP Stack intercept scripts (frida_intercept_engine)
  4. Other useful files in this repo: accept_all_certs.js and the sample apps under example_app/fluttida.
