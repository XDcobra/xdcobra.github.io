2. Overview

Voice Lab - Offline Tools is an on-device speech toolkit for exploring features such as speech-to-text, text-to-speech, voice activity detection, diarization, enhancement, and separation. You can use the app without registration and without creating a user account. This privacy policy explains which data is processed when you use the app and for which purposes.

The app does not provide an account system or user profiles. We do not assign you a user ID. Third-party advertising SDKs may process device identifiers such as the Advertising ID as described below.

3. Data collection and processing

3.1 Data processed locally by the app

The app processes the following data locally on your device:

  • Audio files or recordings you provide: Audio is processed on-device for speech tasks such as STT, TTS, VAD, diarization, enhancement, and separation.
  • Model files and metadata: Downloaded model files, checksums, and local manifest data (e.g. download timestamp, last-used timestamp) are stored locally to manage offline use.
  • Model settings and app preferences: Selected models, decoding parameters, and UI preferences are stored locally.

These locally stored data are not accessed by us as the provider and are not transmitted to our own servers. We only process personal data if you contact us directly, for example via email.

3.3 Model downloads from GitHub Releases

The app lets you download speech models directly from the official sherpa-onnx GitHub Releases page. These downloads are initiated by you and stored locally on your device so the models can be used offline.

Data that may be processed when downloading models:

  • IP address
  • Device and network information (e.g. device type, operating system, connection type)
  • Download metadata (e.g. file requested, timestamp, size)

To verify integrity, the app may compute cryptographic checksums (SHA-256) of downloaded model files on your device. These checksum calculations are performed locally and are not transmitted to our servers.

Downloads are served by GitHub (GitHub, Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA) or its infrastructure providers. GitHub may process the data above in accordance with its privacy policy.

GitHub privacy information:
https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement
